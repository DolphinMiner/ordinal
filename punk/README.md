# Punk backend

## 环境配置

### 1.Windows
在`C:\Users\Your-Username\.aws\credentials`文件中配置 AWS 访问凭证：
```
[default]
aws_access_key_id = <ACCESS_KEY_ID>
aws_secret_access_key = <SECRET_ACCESS_KEY>
```

### 2.Linux & Mac 
在 ~/.aws/credentials 文件中配置 AWS 访问凭证：
```
[default]
aws_access_key_id = <ACCESS_KEY_ID>
aws_secret_access_key = <SECRET_ACCESS_KEY>
```
